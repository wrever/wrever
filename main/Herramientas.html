Owasp ZAP
Zed Attack Proxy

Esta es una de las herramientas más completas a la hora de iniciar una auditoria de seguridad. ZAP realiza pruebas de seguridad, incluso si usted no tiene experiencia en pruebas de seguridad, ya que, la herramienta esta totalmente automatizada para ello. Tipos de testeos:
Vulnerability Assessment – The system is scanned and analyzed for security issues.
Penetration Testing – The system undergoes analysis and attack from simulated malicious attackers.
Runtime Testing – The system undergoes analysis and security testing from an end-user.
Code Review – The system code undergoes a detailed review and analysis looking specifically for security vulnerabilities.

Astra's Pentest Platform
WEB APP PENTEST

El escáner de vulnerabilidades de Astra realiza escaneos de vulnerabilidades automatizados impulsados ​​por años de experiencia en inteligencia de seguridad y numerosos datos de escaneos de seguridad. Realiza más de 3000 pruebas para cubrir una amplia gama de CVE, OWASP top 10 y SANS 25. Las empresas pueden integrar el escáner de vulnerabilidades con su pila tecnológica con la función de integración CI/CD (todavía está trabajando para permitir más integraciones con tecnología auxiliar.).
